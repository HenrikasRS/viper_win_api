Prerequisites

The following requirements should be met before starting development with Rubedos Viper.Net:

Minimum Requirements

  • Rubedos VIPER™ Perception System evaluation kit (visit www.rubedos.com to get one)

  • PC machine with i386 or AMD 64 architecture processor

  • MS Windows 7 / 8 / 10

  • 4 GB RAM

  • .Net Framework 4.5.2 or higher

  • MS Visual Studio 2015 or higher (Community, Professional or Enterprise)

Highly Recommended

  • Intel Core i7 processor on Windows 10

  • 16 GB RAM

  • NVidia GPU with CUDA capability 3.5
